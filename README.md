# Entry!

## サイト概要

### サイトテーマ

各スポーツの練習試合の相手を探すサイト

### テーマを選んだ理由

学生時代、部活の練習試合の相手を探すのに苦労したため、練習試合の相手を探すサイトがあれば便利だと考え、このテーマにしました。

### ターゲットユーザ

練習試合相手を探す顧問やチーム

### 主な利用シーン

- 練習試合の相手を募集しているチームを探す時
- 練習試合をしたい相手に申し込む時
